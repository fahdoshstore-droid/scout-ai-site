## 🎯 عن المشروع

SCOUT AI منصة ذكاء اصطناعي لاكتشاف وتقييم مواهب كرة القدم الناشئين في السعودية.

المنصة تجمع **الأكاديميات** و**اللاعبين** و**الكشافين** في نظام واحد — كل طرف بحسابه يشوف اللي يخصه.

## ✨ المميزات

- 🧠 **تحليل AI حقيقي** — Anthropic Vision API
- ⚖️ **Scoring Engine** — أوزان مختلفة لكل مركز
- 🧬 **Sport DNA** — يكتشف المركز الأنسب لكل لاعب
- 📋 **تقييم كشاف مهيكل** — فورم احترافي موحد
- 🔄 **دمج ذكي** — AI 40% + كشاف 60%
- 📊 **معايير FIFA** — مقارنة عمرية دولية
- ⭐ **لاعب مشابه** — يقارنك بلاعب محترف
- 💪 **توصيات تدريبية** — مخصصة من AI

## 🏗️ البنية التقنية

- **Frontend:** React PWA
- **AI Engine:** Anthropic Vision API (Claude)
- **Scoring:** Position-weighted algorithm (6 positions × 8 skills)
- **Sport DNA:** Multi-position compatibility algorithm
- **Backend (Roadmap):** Node.js + PostgreSQL
